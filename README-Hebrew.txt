# משחק ראש רץ - Headrunner
## גרסה מתורגמת לעברית

### מה שונה בגרסה זו:
1. **תרגום לעברית**: כל הטקסטים במשחק תורגמו לעברית
2. **תמיכה ב-RTL**: הטקסט מוצג בכיווניות מימין לשמאל
3. **ללא לינקים חיצוניים**: הוסרו כל הקישורים החיצוניים

### רשימת התרגומים שבוצעו:
- "Start Game" → "התחל משחק"
- "Game Over" → "משחק הסתיים"
- "You Died" → "הפסדת"
- "Confirm" → "אישור"
- "Clear" → "נקה"
- "Enter Answer Quickly!" → "הכנס תשובה מהר!"
- "Timer" → "זמן"
- "Restart" → "התחל מחדש"
- "End Game" → "סיום משחק"
- "Ok" → "אישור"
- "Give Me 3 crystals and I unlock the door" → "תן לי 3 גבישים ואפתח את הדלת"
- "Banana!!! Come in Please" → "בננה!!! בואי בבקשה"
- "Arggh!!! Go Away!" → "אררר!!! לך מכאן!"
- "Doors to new level is opened!" → "הדלת לשלב החדש נפתחה!"
- "Items:" → "פריטים:"
- "Level 1 Complete" → "שלב 1 הושלם"
- "Start Next Level" → "התחל שלב הבא"

### איך להפעיל את המשחק:

#### אפשרות 1: באמצעות Python
1. וודאי שמותקן Python 3.x במחשב
2. פתחי Command Prompt או PowerShell
3. נווטי לתיקיית המשחק:
   ```
   cd path\to\headrunner-master
   ```
4. הפעילי שרת HTTP מקומי:
   ```
   python -m http.server 8888
   ```
5. פתחי דפדפן וגשי לכתובת:
   ```
   http://localhost:8888/game.html
   ```

#### אפשרות 2: באמצעות שרת אחר
אם יש לך שרת אינטרנט מקומי אחר (כמו XAMPP, WAMP, וכו'), העתיקי את כל הקבצים לתיקיית השרת ופתחי את game.html.

### קבצים חשובים:
- `game.html` - קובץ ה-HTML הראשי למשחק
- `static/` - תיקייה עם כל קבצי ה-JavaScript וה-CSS
- `data/` - תיקייה עם נתוני המשחק (מפות, ספרייטים)
- `images/` - תיקייה עם תמונות המשחק

### הערות טכניות:
- המשחק דורש שרת HTTP בגלל הגבלות CORS של הדפדפן
- לא ניתן לפתוח את game.html ישירות מהתיקייה (file://)
- המשחק נבדק ועובד בדפדפנים מודרניים

### איך לשחק:
- השתמשו בחצים כדי להזיז את הדמות
- איספו גבישים (3 בסך הכל)
- הימנעו מאויבים
- פתרו חידות מתמטיות כדי להתקדם
- הגיעו לדלת כדי לעבור לשלב הבא

### בקרים:
- חץ שמאל/ימין: תנועה
- חץ עליון או X: קפיצה
- עכבר: לחיצה על כפתורים ופתרון חידות
